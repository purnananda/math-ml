          1. Basic Arithmetic Level 1

          Arithmetic is the language of numbers. Every ML model — no matter how fancy — is ultimately doing arithmetic on numbers, just a lot of it. Fluent arithmetic means later formulas won't feel intimidating.

          1.1 Order of Operations (PEMDAS / BODMAS)

          When an expression has several operations, you can't just work left-to-right. There is an agreed order:

          1. Parentheses (brackets)
          2. Exponents (powers, roots)
          3. Multiplication and Division (left to right)
          4. Addition and Subtraction (left to right)
          Example
          $3 + 4 \times 2 = 3 + 8 = 11$  (not $14$).
          $(3 + 4) \times 2 = 14$.
          $2 + 3^2 \times 4 = 2 + 9 \times 4 = 38$.

          1.2 Negative Numbers

          Key rules:

          • $-a + a = 0$
          • $(-a) \times (-b) = a \times b$ (negative × negative = positive)
          • $(-a) \times b = -(a \times b)$

          1.3 Fractions

          A fraction $\frac{a}{b}$ represents division into $b$ equal parts.

          • Add / subtract: same denominator required. $\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}$.
          • Multiply: $\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}$.
          • Divide: flip the second. $\frac{a}{b} \div \frac{c}{d} = \frac{ad}{bc}$.
          Example
          $\frac{2}{3} + \frac{1}{4} = \frac{8}{12} + \frac{3}{12} = \frac{11}{12}$.

          1.4 Decimals & Percentages

          $37\% = \frac{37}{100} = 0.37$. Percent change: $\frac{\text{new} - \text{old}}{\text{old}} \times 100$.

          Example
          A stock from $\$40$ to $\$46$: $\frac{6}{40} \times 100 = 15\%$ increase.

          1.5 Ratios & Proportions

          Proportion: $\frac{a}{b} = \frac{c}{d}$. Cross-multiply: $ad = bc$.

          Example
          If 3 apples cost $\$1.20$, 8 apples cost $\frac{8 \times 1.20}{3} = \$3.20$.

          1.6 Exponents

          RuleExample
          $x^a \cdot x^b = x^{a+b}$$2^3 \cdot 2^4 = 2^7 = 128$
          $\frac{x^a}{x^b} = x^{a-b}$$\frac{5^6}{5^2} = 5^4$
          $(x^a)^b = x^{ab}$$(3^2)^3 = 3^6$
          $x^0 = 1$,   $x^{-n} = \frac{1}{x^n}$$2^{-3} = \frac{1}{8}$

          1.7 Square Roots

          $\sqrt{x}$ is the non-negative number that squares to $x$. Related: $\sqrt{x} = x^{1/2}$.

          🎯 Quick Quiz

          1. What is $12 - 3 \times 2 + 4^2$?
          2. What is 18% of 250?
          3. Simplify $\frac{2^5 \cdot 2^3}{2^4}$.

          2. Algebra Level 1

          Algebra generalizes arithmetic with letters (variables) for unknowns. Every ML model is an algebraic relationship — algebra is how we write the model.

          2.1 Variables, Constants, Coefficients

          • Variable: a letter for an unknown ($x, y$).
          • Constant: a fixed number ($3, \pi$).
          • Coefficient: the number multiplying a variable ($5$ in $5x$).

          2.2 Combining Like Terms

          Example
          $3x + 5x - 2x = 6x$. But $3x + 5x^2$ can't be combined.

          2.3 Distributive Property

          $a(b + c) = ab + ac$ — one of the most-used rules in all of math.

          Example
          $3(x + 4) = 3x + 12$.   $-2(y - 5) = -2y + 10$.

          2.4 Solving Linear Equations

          Goal: isolate the variable. Whatever you do to one side, do to the other.

          Example — multi-step
          Solve $3(x - 2) + 4 = 16$.
          Distribute: $3x - 6 + 4 = 16 \implies 3x - 2 = 16$.
          Add 2: $3x = 18 \implies x = 6$.

          2.5 Rearranging Formulas

          Example
          $y = mx + b$. Solve for $x$: $x = \frac{y - b}{m}$.

          2.6 Systems of Equations

          Example
          $x + y = 10$ and $x - y = 4$. Add: $2x = 14 \implies x = 7$, so $y = 3$.

          2.7 Quadratic Formula

          For $ax^2 + bx + c = 0$:   $x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

          Example
          $x^2 - 5x + 6 = 0$:   $x = \frac{5 \pm \sqrt{25 - 24}}{2} = \frac{5 \pm 1}{2}$. So $x = 3$ or $x = 2$.
          Why this matters for ML
          The cost function for linear regression is a quadratic in the model's parameters. Understanding quadratics = understanding the shape of the "error surface" you'll walk down with gradient descent.

          3. Functions Level 1

          A function is a rule: input goes in → exactly one output comes out. Every ML model is literally a function.

          3.1 Notation

          $f(x)$ = "the output of $f$ when the input is $x$". Domain = valid inputs; Range = possible outputs.

          3.2 Evaluating

          Example
          If $f(x) = x^2 - 3x + 1$: $f(4) = 16 - 12 + 1 = 5$; $f(-2) = 4 + 6 + 1 = 11$.

          3.3 Linear Functions

          $f(x) = mx + b$. Straight line, constant rate of change.

          ML Connection
          Linear regression is literally $\hat{y} = wx + b$. Machine learning trains $w$ and $b$ from data. That's it.

          3.4 Quadratic Functions

          $f(x) = ax^2 + bx + c$. Parabola. If $a > 0$ opens up (has minimum). Vertex at $x = -\frac{b}{2a}$.

          3.5 Function Composition

          $(f \circ g)(x) = f(g(x))$. A neural network is a giant composition — that's what "deep" means.

          3.6 Other Shapes

          • Exponential $e^x$ — grows fast.
          • Logarithm $\log(x)$ — grows slowly.
          • Sigmoid $\sigma(x) = \frac{1}{1 + e^{-x}}$ — squashes any input into $(0, 1)$; used in classification.

          4. Graphs Level 1

          A graph turns a function into a picture. Seeing math visually accelerates intuition.

          4.1 The Cartesian Plane

          • Two axes meet at the origin $(0, 0)$.
          • X-axis: horizontal (input). Y-axis: vertical (output).
          • A point is $(x, y)$.

          4.2 Slope

          $m = \dfrac{y_2 - y_1}{x_2 - x_1}$ — rise over run.

          • $m > 0$: goes up-right. $m < 0$: down-right. $m = 0$: horizontal.
          Example
          Points $(1, 2)$ and $(5, 10)$: $m = \frac{8}{4} = 2$.

          4.3 Slope-Intercept Form

          $y = mx + b$: $m$ = slope, $b$ = y-intercept (where the line crosses $y$-axis).

          ML Connection
          Cost functions in ML are curves (usually valleys). Training = "rolling downhill". The slope at your spot tells you which direction to move.

          5. Summation (Σ) Level 2

          Sigma ($\Sigma$) is a compact way to write "add up a bunch of things". Nearly every ML formula uses it.

          5.1 The Notation

          $\displaystyle\sum_{i=1}^{n} x_i$ means "start with $i=1$, add $x_i$, increase $i$ by 1, repeat until $i=n$". It's a for-loop that adds.

          Example
          $\sum_{i=1}^{5} i = 1 + 2 + 3 + 4 + 5 = 15$.
          Example
          $\sum_{i=1}^{4} i^2 = 1 + 4 + 9 + 16 = 30$.

          5.2 Properties

          • Constants pull out: $\sum c x_i = c \sum x_i$.
          • Sum of a constant: $\sum_{i=1}^{n} c = nc$.
          • Splits over addition: $\sum (x_i + y_i) = \sum x_i + \sum y_i$.

          5.3 Mean Squared Error (MSE)

          $\text{MSE} = \dfrac{1}{n} \displaystyle\sum_{i=1}^{n} (\hat{y}_i - y_i)^2$ — the classic ML loss function.

          ML Connection
          Whenever you see a Greek $\Sigma$ in an ML paper, whisper: "it's a for-loop that adds". You'll unlock 80% of formulas.

          🎯 Quick Quiz

          1. Compute $\sum_{i=1}^{6} i$.
          2. Given predictions $\hat{y}=[3,5,8]$ and true $y=[2,5,10]$, what is MSE?

          6. Averages Level 2

          "Average" isn't one thing — it's a family of tools for summarizing many values into one number.

          6.1 Mean

          $\bar{x} = \dfrac{1}{n}\sum x_i$. Best for symmetric data. Weakness: sensitive to outliers.

          6.2 Median

          Sort; take the middle value. Robust to outliers.

          6.3 Mode

          Most frequent value. Useful for categorical data.

          6.4 Weighted Average

          $\bar{x}_w = \dfrac{\sum w_i x_i}{\sum w_i}$.

          Example — Course grade
          Quizzes 88 (20%), midterm 75 (30%), final 92 (50%): $\bar{x}_w = 0.2(88) + 0.3(75) + 0.5(92) = 86.1$.

          6.5 Compare

          Data: $[1, 2, 2, 3, 100]$. Mean = 21.6 (skewed), Median = 2, Mode = 2.

          ML Connection
          The mean minimizes squared error (MSE). The median minimizes absolute error (MAE). Different losses → different "averages".

          🎯 Quick Quiz

          1. Median of $[3, 8, 10, 12, 15]$?
          2. Weighted average of $[80, 90, 70]$ with weights $[0.5, 0.3, 0.2]$?

          7. Probability Level 2

          Probability quantifies uncertainty. ML models often output probabilities: "80% chance this email is spam".

          7.1 Fundamentals

          $P(A) = \dfrac{\text{favorable outcomes}}{\text{total outcomes}}$. Range: $[0, 1]$.

          7.2 Complement Rule

          $P(\text{not } A) = 1 - P(A)$.

          7.3 Independent Events (AND)

          $P(A \text{ and } B) = P(A) \cdot P(B)$ if independent.

          Example
          3 heads in a row: $(1/2)^3 = 1/8$.

          7.4 Addition Rule (OR)

          $P(A \text{ or } B) = P(A) + P(B) - P(A \text{ and } B)$.

          7.5 Conditional Probability

          $P(A \mid B) = \dfrac{P(A \text{ and } B)}{P(B)}$.

          7.6 Bayes' Theorem

          $P(A \mid B) = \dfrac{P(B \mid A) \cdot P(A)}{P(B)}$ — flip a conditional. Used in spam filters, medical tests, Naive Bayes.

          ML Connection
          Logistic regression outputs $P(y=1 \mid x)$. Softmax gives class probabilities. Naive Bayes applies Bayes' theorem directly.

          🎯 Quick Quiz

          1. Two dice — probability both show 6?
          2. Two rolls — probability of at least one 6?

          8. Statistics Level 2

          Statistics is the science of learning from data. ML is applied statistics with more compute.

          8.1 Population vs Sample

          Population = everything. Sample = subset you measured. We estimate population properties from samples.

          8.2 Variance

          Population: $\sigma^2 = \frac{1}{n}\sum (x_i - \mu)^2$. Sample: $s^2 = \frac{1}{n-1}\sum (x_i - \bar{x})^2$.

          8.3 Standard Deviation

          $\sigma = \sqrt{\sigma^2}$. Same units as data → interpretable.

          8.4 Normal Distribution

          Bell curve. 68–95–99.7 rule: ~68% within $\pm 1\sigma$, ~95% within $\pm 2\sigma$, ~99.7% within $\pm 3\sigma$.

          8.5 Z-Score

          $z = \dfrac{x - \mu}{\sigma}$ — how many std devs from the mean.

          Example
          Class avg 70, $\sigma$ 10. You score 85. $z = 1.5$ → top ~7%.

          8.6 Correlation

          From $-1$ to $+1$. Correlation ≠ causation!

          ML Connection
          Preprocessing usually standardizes features to zero mean, unit variance (z-scores). Distributions matter — many models assume roughly Gaussian data.

          🎯 Quick Quiz

          1. Scores normal, $\mu=75, \sigma=8$. What % scored between 67 and 83?
          2. Your height 180 cm; mean 172, $\sigma$ 7. Z-score?

          9. Vectors Level 3

          A vector is an ordered list of numbers. In ML, a data example is a vector.

          9.1 Representation

          $\mathbf{v} = \begin{bmatrix} 2 \\ 5 \\ 3 \end{bmatrix}$ or $(2, 5, 3)$. Dimension = count. Geometrically, an arrow from origin.

          9.2 Scalar Multiplication

          $3 \cdot (2, -1, 4) = (6, -3, 12)$. Scales the arrow.

          9.3 Addition / Subtraction

          Element-wise. $(1, 2, 3) + (4, 5, 6) = (5, 7, 9)$. Geometrically: tip-to-tail.

          9.4 Magnitude

          $\|\mathbf{v}\| = \sqrt{v_1^2 + \cdots + v_n^2}$. $\|(3, 4)\| = 5$.

          9.5 Unit Vectors

          $\hat{\mathbf{v}} = \dfrac{\mathbf{v}}{\|\mathbf{v}\|}$. Length 1, same direction.

          9.6 Dot Product

          $\mathbf{a} \cdot \mathbf{b} = \sum a_i b_i = \|\mathbf{a}\| \|\mathbf{b}\| \cos\theta$.

          • Positive → similar directions. Zero → perpendicular. Negative → opposite.

          9.7 Cosine Similarity

          $\cos\theta = \dfrac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}$ — the similarity metric in NLP & recommenders.

          ML Connection
          Prediction in linear regression: $\hat{y} = \mathbf{w} \cdot \mathbf{x} + b$. In embeddings, "king − man + woman ≈ queen" is vector arithmetic.

          10. Matrices Level 3

          A rectangular grid of numbers. Often: rows = examples, columns = features.

          10.1 Anatomy

          $A = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix}$ — a $2 \times 3$ matrix. $a_{ij}$ = row $i$, column $j$.

          10.2 Addition

          Element-wise, same shape required.

          10.3 Multiplication

          $C = AB$: $A$ is $m \times k$, $B$ is $k \times n$, $C$ is $m \times n$. Entry $C_{ij}$ = dot of row $i$ of $A$ with column $j$ of $B$.

          Example
          $\begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix} \begin{bmatrix} 5 & 6 \\ 7 & 8 \end{bmatrix} = \begin{bmatrix} 19 & 22 \\ 43 & 50 \end{bmatrix}$
          Not commutative! Usually $AB \neq BA$.

          10.4 Transpose

          $A^T$: rows become columns. $(AB)^T = B^T A^T$.

          10.5 Identity

          $I$ has 1s on diagonal. $AI = A$.

          ML Connection
          A batch of $n$ examples with $d$ features is an $n \times d$ matrix $X$. Predictions: $\hat{\mathbf{y}} = X\mathbf{w} + b$. Neural net layer: $\mathbf{a}_{\text{next}} = \sigma(W\mathbf{a} + \mathbf{b})$. GPUs = giant matrix multipliers.

          11. Linear Algebra Level 3

          Linear algebra pulls vectors and matrices together. It's the language ML thinks in.

          11.1 Linear Combinations

          $c_1 \mathbf{v}_1 + c_2 \mathbf{v}_2 + \cdots$ — scale each vector, add them up.

          11.2 Span

          The set of all linear combinations of a group of vectors. One vector → line. Two non-parallel → plane. Three non-coplanar → 3D space.

          11.3 Linear Independence

          Vectors where none can be written as a combination of the others.

          11.4 Matrix as Transformation

          Multiplying by a matrix moves every input vector to a new output — rotations, reflections, scalings, projections.

          11.5 Determinant (Preview)

          Measures how much a matrix scales area/volume. If $\det = 0$, the transformation squashes to lower dimension.

          11.6 Eigenvalues / Eigenvectors (Preview)

          Special vectors that only get scaled by a matrix: $A\mathbf{v} = \lambda \mathbf{v}$. Foundation of PCA, PageRank, spectral methods.

          ML Connection
          • Every dataset is a matrix.
          • PCA finds eigenvectors of the covariance matrix.
          • Neural net layers = matrix multiplication + activation.
          Practical priority for beginner ML: dot products, matrix–vector multiplication, transpose. Master those first; treat eigenvalues as "know they exist".

          12. Calculus (Derivatives) Level 4

          Calculus is the mathematics of change. In ML we ask: "if I nudge this parameter, how does the loss change?" That's a derivative.

          12.1 Derivative Meaning

          $f'(x)$ = instantaneous slope of $f$ at $x$. How fast is $f$ changing right here?

          12.2 Power Rule

          $\dfrac{d}{dx} x^n = n x^{n-1}$. The most-used derivative rule.

          Example
          $\frac{d}{dx} x^5 = 5x^4$. $\frac{d}{dx} \sqrt{x} = \frac{1}{2\sqrt{x}}$.

          12.3 Other Rules

          RuleFormula
          Constant$\frac{d}{dx} c = 0$
          Constant multiple$\frac{d}{dx}[cf] = cf'$
          Sum$(f+g)' = f' + g'$
          Product$(fg)' = f'g + fg'$
          $e^x$$\frac{d}{dx} e^x = e^x$
          $\ln x$$\frac{d}{dx} \ln x = \frac{1}{x}$
          Example
          $f(x) = 4x^3 - 7x^2 + 2x - 9 \implies f'(x) = 12x^2 - 14x + 2$.

          12.4 Partial Derivatives

          Multi-input function: differentiate w.r.t. one variable, hold others constant. Notation: $\dfrac{\partial f}{\partial x}$.

          Example
          $f(x, y) = 3x^2 y + 4xy^3$.
          $\frac{\partial f}{\partial x} = 6xy + 4y^3$,   $\frac{\partial f}{\partial y} = 3x^2 + 12xy^2$.

          12.5 Chain Rule

          $\dfrac{d}{dx} f(g(x)) = f'(g(x)) \cdot g'(x)$. This is backpropagation.

          Example
          $f(x) = (3x + 1)^4 \implies f'(x) = 4(3x+1)^3 \cdot 3 = 12(3x+1)^3$.

          12.6 The Gradient

          $\nabla f = \left( \frac{\partial f}{\partial x_1}, \dots, \frac{\partial f}{\partial x_n} \right)$. Points in the direction of steepest increase.

          ML Connection
          Training = compute loss → compute gradient (chain rule = backprop) → step opposite to gradient. Master the chain rule and you'll master neural nets.

          13. Gradient Descent Level 4

          Gradient descent is the optimization algorithm behind almost every ML model. Uses the gradient to walk downhill on a cost function.

          13.1 The Cost Function

          $J(w, b) = \dfrac{1}{2n} \sum (\hat{y}_i - y_i)^2$ where $\hat{y}_i = wx_i + b$. Lower = better.

          13.2 The Big Picture

          You're in a foggy hilly landscape. You want the valley. Strategy: feel the slope → step downhill → repeat. That's it.

          13.3 Update Rule

          $\theta \leftarrow \theta - \alpha \cdot \dfrac{\partial J}{\partial \theta}$. $\alpha$ = learning rate.

          For linear regression:

          $\dfrac{\partial J}{\partial w} = \dfrac{1}{n} \sum (\hat{y}_i - y_i) x_i, \quad \dfrac{\partial J}{\partial b} = \dfrac{1}{n} \sum (\hat{y}_i - y_i)$

          13.4 Learning Rate

          • Too small: slow.
          • Too big: overshoots, diverges.
          • Just right: fast, stable.

          13.5 Worked Mini-Example

          Example
          Data: $(x, y) = (2, 5)$. Start $w=1, b=0$, $\alpha=0.1$.
          Predict: $\hat{y} = 2$. Error: $-3$.
          Gradients: $\partial w = -6$, $\partial b = -3$.
          Update: $w = 1.6$, $b = 0.3$. New prediction: $3.5$. Closer to $5$!

          13.6 Local vs Global Minimum

          • Linear regression: convex → always global.
          • Neural nets: many minima, but most "good enough" ones work.

          13.7 Variants

          • Batch GD: full dataset per step. Accurate but slow.
          • SGD: one example. Noisy but fast.
          • Mini-batch SGD: 32–256 examples. Deep learning default.
          • Adam, Momentum, RMSProp: smart enhancements.
          The Grand Summary
          1. Initialize parameters randomly.
          2. Compute predictions and loss.
          3. Compute gradient (backprop).
          4. Update: $\theta \leftarrow \theta - \alpha \nabla J$.
          5. Repeat.
          Every model trains this way. You now understand the heart of ML. 🎉
